安装环境

三台机器分别是 hadoop102，hadoop103,hadoop104

安装epel-release

Extra Packages for Enterprise Linux是为“红帽系”的操作系统提供额外的软件包，适用于RHEL、CentOS和Scientific Linux。相当于是一个软件仓库，大多数rpm包在官方 repository 中是找不到的）

关闭防火墙，关闭防火墙开机自启

创建 hadoop 用户和 bigdata 用户组

配置 hadoop 用户具有 root 权限，方便后期加 sudo 执行 root 权限的命令（暂时）

修改/etc/sudoers 文件，在 %wheel 这行下面添加一行，如下

第一行: 允许用户 youuser执行sudo命令(需要输入密码).
第二行: 允许用户组youuser里面的用户执行sudo命令(需要输入密码).
第三行: 允许用户youuser执行sudo命令,并且在执行的时候不输入密码.
第四行: 允许用户组youuser里面的用户执行sudo命令,并且在执行的时候不输入密码.

创建文件夹

为了方便管理，在系统的根目录下 使用 root 用户创建 hadoop 目录， 并将 hadoop 目录权限给 hadoop 用户，再使用 hadoop 用户在 hadoop 目录下分别创建 module，software 目录。注意创建目录使用 hadoop 用户进行创建，如果使用 root 创建，需要将 hadoop 目录的属主和属主组进行变更

安装 jdk

将jdk17 上次到/hadoop/software 目录，并解压迁移到 module 目录，设置环境变量。
